## O Início de Tudo

Programar para **SNES** não é como programar em JavaScript. Não existe `console.log`.

Para pintar um pixel na tela, você precisa falar diretamente com a PPU (Picture Processing Unit).

```assembly
; Exemplo simples de código 65816
LDA #$80    ; Carrega o valor hex 80 no acumulador
STA $2100   ; Armazena no registrador de brilho da tela (INIDISP)
```
